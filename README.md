# floorplanner

## Card Show Nation target

Set `NEXT_PUBLIC_CSN_BASE_URL` to the Card Show Nation environment you want Floorplanner to point at.

- Private tunnel example: `https://nasty-zebras-lead.loca.lt`
- Local Card Show Nation dev server: `http://localhost:3000`
- Eventual public domain: `https://cardshownation.com`

Floorplanner now normalizes that value and exposes it in the status bar so you can verify which Card Show Nation environment the app is targeting.
